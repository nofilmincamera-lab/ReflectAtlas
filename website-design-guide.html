Design Principles

1. Geometric Simplicity

Use clean shapes, bold colors, and minimal realism. Every element should be reduced to its essential form—"count the wings, not the feathers." Favor flat design with hard edges over gradients and shadows.

2. Breathing Room

Generous whitespace creates clarity and sophistication. Never crowd elements. Use asymmetric layouts that give each component space to breathe and draw the eye naturally through the page.

3. Bold Color as Structure

Large expanses of solid brand colors define sections and create visual hierarchy. Use color intentionally—not decoratively. Each color block should serve a functional purpose in guiding the user.

4. Functional Animation

Motion should feel purposeful, never gratuitous. Use subtle transitions to provide feedback, guide attention, and enhance usability. Keep animations short (200-400ms) and use easing functions.

Color Blocking

Bold color blocks create impact

Use large solid-color sections to define different content areas and create visual rhythm throughout the page.

Best Practice: Use asymmetric color blocks to break up long pages. Alternate between light backgrounds (#F5F3EF) and dark backgrounds (#2D2420) to create visual sections. Avoid using more than 2-3 brand colors in a single section.

Accessibility & Performance

Body text #2D2420 on #FFFFFF 13.5:1 AAA

Primary button #FFFFFF on #DC2626 5.8:1 AA

WCAG 2.1 AA Compliance: All text must have a contrast ratio of at least 4.5:1 (3:1 for large text 18px+). Test all color combinations with a contrast checker before implementation.

Performance Guidelines

Page Load Optimization

  • Target: First Contentful Paint under 1.8s
  • Images: Use WebP format, lazy load below fold
  • Fonts: Subset fonts, use font-display: swap
  • CSS: Critical CSS inline, defer non-critical
  • JavaScript: Code split, defer non-essential scripts
  • SVG: Inline small icons, external large illustrations

Keyboard Navigation

/* Ensure visible focus states */ a:focus, button:focus, input:focus { outline: 3px solid #3B82F6; outline-offset: 2px; } /* Skip to main content link */ .skip-link { position: absolute; top: -40px; left: 0; background: #DC2626; color: white; padding: 8px 16px; text-decoration: none; border-radius: 0 0 4px 0; } .skip-link:focus { top: 0; }

Motion Preferences

/* Respect prefers-reduced-motion */ @media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } }
